Four ways to bring in data

Every tab below accepts a plain CSV or an Excel (.xlsx) file — upload your working spreadsheet as-is, no need to save as CSV first.

Leads Upload a CRM export of your leads — name, source, campaign, stage/status, project.
Ad Spend Upload a Google Ads / Meta Ads / LinkedIn campaign performance export as-is — pick the platform and project once, the file supplies the rest.
Bookings Upload closed deals (revenue & GTV) to match against leads already in the system — this is what powers realized ROAS.
Campaign Summary Upload a per-campaign performance sheet (Spend, Leads, Potential, Hot/Super Hot, Site Visits) — powers the Campaign Performance table, including Opportunity Cost and marginal metrics.

Expected columns (accepted alternate headers in brackets): name, source [platform, channel], campaign_name [campaign], created_date [date], current_stage [stage], lead_status [status], project [property] (optional — falls back to the Project selected below). source must be one of Google / Meta / LinkedIn / WhatsApp/SMS / Alldoors Website / Data Calling Team.
